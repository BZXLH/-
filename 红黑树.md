# 红黑树

**定义：**

是一种平衡二叉搜索树（最长路径不超过最短路径两倍）

**特点：**

1. 每个节点，非黑即红
2. 根节点为黑，叶节点为黑
3. 不能存在连续的两个红节点
4. 任何节点，至其下属的、不同的叶节点的每条路径上，黑节点数必须相等

**操作：**

- 旋转（与一般平衡二叉搜索树相同）
- 变色
